Welcome zu MadSonic

Mit MadSonic können Sie ganz einfach auf Ihren Android Handy Media Dateien von Ihrem Computer zu Hause streamen oder herunterladen.

     Standardmäßig ist dieses Programm so konfiguriert, um den MadSonic Server verwenden . Sie können aber Ihren eigenen Server eingestellen, gehen Sie dafür bitte auf Einstellungen und die Konfiguration zu ändern.

     Sie können dieses Programm kostenlos zu nutzen.
